============
Users Manual
============

Target group: **Editors**

#. Select a page where you want to insert schooldirectory

#. Insert a new content element of type: `Insert Plugin`

#. On tab `Plugin` select `Schooldirectory` from selectbox.

#. Confirm reload to load special input fields for our extension (FlexForm).

#. Save

School Record
-------------

#. Select a Storage Page

#. Create a new record of type `School`

#. Give it a title.

#. Enter a description which will be shown on configured detail page

#. By default you can insert up to 5 images to schoole record.
